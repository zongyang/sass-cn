# 安装

有很多应用程序可以让您在几分钟内启动并运行Sass，适用于Mac、Windows和Linux。您可以免费下载大部分应用程序，但也有一些是付费应用程序 (但完全值得)。

  • CodeKit (Paid) Mac
  • Compass.app (Paid, Open Source) Mac Windows Linux
  • Ghostlab (Paid) Mac Windows
  • Hammer (Paid) Mac
  • Koala (Open Source) Mac Windows Linux
  • LiveReload (Paid, Open Source) Mac Windows
  • Prepros (Paid) Mac Windows Linux
  • Scout-App (Free, Open Source) Windows Linux Mac

# 命令行

当您使用命令行安装Sass时，您将能够运行Sass把.sass和.scss文件编译为.css文件。例如：

sass source/stylesheets/index.scss build/stylesheets/index.css

首先使用下面的某个选项安装SASS，然后运行sass --version以确保它安装正确。如果有，将包括1.22.9版本。您还可以运行sass--help获取有关命令行界面的更多信息。

如果一切就绪，就去尽情体验吧。如果您是初来乍到，我们已经建立了一些资源来帮助您快速学习。

# 跨平台（独立安装）

您可以在Windows、Mac或Linux上安装Sass，通过在GitHub下载相应操作系统的包，并配置好环境变量PATH。没有外部依赖，也不需要安装其他任何东西。

# 跨平台（npm安装）

如果您使用Node.js，可以通过npm运行以下命令来安装Sass：

npm install -g sass

但是，请注意，这种方式安装的Sass是以纯JavaScript实现的，它的运行速度比这里列出的其它方式稍微慢一些。但是它有相同的接口，所以如果您需要更快的速度，稍后可以很容易地切换到另一个环境中运行！

# Windows（Chocolatey安装）

如果您使用的是Windows版本的Chocolatey包管理器，可以通过以下命令安装Dart Sass：

choco install sass

# Mac OS X（Homebrew安装）

如果您的操作系统是Mac OS X，可以通过Homebrew包管理器，可以通过以下命令安装Dart Sass：

brew install sass/sass/sass
