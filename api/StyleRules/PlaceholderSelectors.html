# 占位符选择器

Sass有一种特殊的选择器，称为“占位符”。它看起来和操作都很像一个类选择器，但是它以%开头，并且不包含在CSS输出中。事实上，任何复杂的选择器（如逗号之间的选择器）若包含占位符的选择器将不会在CSS输出中。

.alert:hover, %strong-alert {
  font-weight: bold;
}

%strong-alert:hover {
  color: red;
}

.alert:hover, %strong-alert
  font-weight: bold


%strong-alert:hover 
  color: red


.alert:hover {
  font-weight: bold;
}

在CSS中不输出，那它有何作用？它可以用于扩展！与类选择器不同，如果没有扩展占位符，占位符不会打乱CSS，也不会强制库的用户为其HTML使用特定的类名。

%toolbelt {
  box-sizing: border-box;
  border-top: 1px rgba(#000, .12) solid;
  padding: 16px 0;
  width: 100%;

  &:hover { border: 2px rgba(#000, .5) solid; }
}

.action-buttons {
  @extend %toolbelt;
  color: #4285f4;
}

.reset-buttons {
  @extend %toolbelt;
  color: #cddc39;
}

%toolbelt
  box-sizing: border-box
  border-top: 1px rgba(#000, .12) solid
  padding: 16px 0
  width: 100%

  &:hover
    border: 2px rgba(#000, .5) solid

.action-buttons
  @extend %toolbelt
  color: #4285f4

.reset-buttons
  @extend %toolbelt
  color: #cddc39

.action-buttons, .reset-buttons {
  box-sizing: border-box;
  border-top: 1px rgba(0, 0, 0, 0.12) solid;
  padding: 16px 0;
  width: 100%;
}
.action-buttons:hover, .reset-buttons:hover {
  border: 2px rgba(0, 0, 0, 0.5) solid;
}

.action-buttons {
  color: #4285f4;
}

.reset-buttons {
  color: #cddc39;
}

在编写Sass库时，占位符选择器非常有用，其中有些规则可以使用也可以不使用。根据经验，如果您只是为自己的应用程序编写样式表，通常最好扩展一个可用的类选择器。
