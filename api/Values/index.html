# 概述

Sass支持多种值类型，其中大多数直接来自CSS。每个表达式都产生一个值，变量保存值。大多数值类型直接来自CSS：

  • 数字（Numbers），可能有单位，也可能没有单位，比如12或100px。
  • 字符串（Strings），可能有也可能没有引号，比如"Helvetica Neue"或bold。
  • 颜色（Colors），可以通过十六进制表示或名称引用，比如#c6538c或blue，或者从函数返回，比如rgb(107、113、127)或hsl(210、100%、20%)。
  • 值列表（Lists），可以用空格或逗号分隔，也可以用方括号括起来，也可以完全不括起来，比如1.5em 1em 0 2em、Helvetica,Arial,sans-serif或[col1-start]。

还有一些是Sass特有的：

  • 布尔值，true和false。
  • null。
  • Maps，如("background": red, "foreground": pink)。
  • get-function()返回的函数引用，并用call()调用。
