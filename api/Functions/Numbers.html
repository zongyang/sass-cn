# 数字函数

# abs()

abs($number) //=> number

返回$number的绝对值。

@debug abs(10px); // 10px
@debug abs(-10px); // 10px

@debug abs(10px) // 10px
@debug abs(-10px) // 10px

# ceil()

ceil($number) //=> number

向上取整。

@debug ceil(4); // 4
@debug ceil(4.2); // 5
@debug ceil(4.9); // 5

debug ceil(4)  // 4
@debug ceil(4.2)  // 5
@debug ceil(4.9)  // 5

# comparable()

comparable($number1, $number2) //=> number

判断$number1和$number2的单位是否兼容。 如果返回true，则$number1和$number2可以安全地进行加法、减法和比较操作。否则，将会产生错误。

@debug comparable(2px, 1px); // true
@debug comparable(100px, 3em); // false
@debug comparable(10cm, 3mm); // true

@debug comparable(2px, 1px)  // true
@debug comparable(100px, 3em)  // false
@debug comparable(10cm, 3mm)  // true

# floor()

floor($number) //=> number

向下取整。

@debug floor(4); // 4
@debug floor(4.2); // 4
@debug floor(4.9); // 4

@debug floor(4)  // 4
@debug floor(4.2)  // 4
@debug floor(4.9)  // 4

# max()

max($number...) //=> number

返回一个或多个数中，最大的数。数字列表可使用...。

@debug max(1px, 4px); // 4px

$widths: 50px, 30px, 100px;
@debug max($widths...); // 100px

@debug max(1px, 4px)  // 4px

$widths: 50px, 30px, 100px 
@debug max($widths...)  // 100px

# min()

min($number...) //=> number

返回一个或多个数中，最小的数。数字列表可使用...。

@debug min(1px, 4px); // 1px

$widths: 50px, 30px, 100px;
@debug min($widths...); // 30px

@debug min(1px, 4px)  // 1px

$widths: 50px, 30px, 100px 
@debug min($widths...)  // 30px

# percentage()

percentage($number) //=> number

将无单位的$number（通常是0到1之间的小数）转换为百分数。

percentage()函数与$number * 100%的结果是相同的。

@debug percentage(0.2); // 20%
@debug percentage(100px / 50px); // 200%

@debug percentage(0.2)  // 20%
@debug percentage(100px / 50px)  // 200%

# random()

random($limit: null) //=> number

如果$limit为null，则返回一个介于0和1之间的随机小数。

@debug random(); // 0.2821251858
@debug random(); // 0.6221325814

@debug random()  // 0.2821251858
@debug random()  // 0.6221325814

如果$limit是一个大于等于1的数，则返回一个介于1和$limit之间的随机数。

@debug random(10); // 4
@debug random(10000); // 5373

@debug random(10)  // 4
@debug random(10000)  // 5373

# round()

round($number) //=> number

四舍五入。

@debug round(4); // 4
@debug round(4.2); // 4
@debug round(4.9); // 5

@debug round(4)  // 4
@debug round(4.2)  // 4
@debug round(4.9)  // 5

# unit()

unit($number) //=> quoted string

返回$number的单位，以带引号的字符串形式表示。

此函数用于调试；它的输出格式不能保证在不同的Sass版本或实现之间保持一致。

@debug unit(100); // ""
@debug unit(100px); // "px"
@debug unit(5px * 10px); // "px*px"
@debug unit(5px / 1s); // "px/s"

@debug unit(100)  // ""
@debug unit(100px)  // "px"
@debug unit(5px * 10px)  // "px*px"
@debug unit(5px / 1s)  // "px/s"

# unitless()

unitless($number) //=> boolean

判断$number是否没有单位。

@debug unitless(100); // true
@debug unitless(100px); // false

@debug unitless(100)  // true
@debug unitless(100px)  // false
