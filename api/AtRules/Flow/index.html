# 概述

Sass提供了许多@规则，这些@规则可以控制样式是被发出，还是多次以较小的变化发出样式。还可以在mixin和函数中编写小型算法，使编写Sass更容易。Sass支持四个流量控制规则：

  • @if控制是否计算块。
  • @each为list中的每个元素或map中的每对元素计算一个块。
  • @for对一个块计算一定次数。
  • @while计算一个块，直到满足某个条件。
