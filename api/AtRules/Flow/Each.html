# @each

@each规则可以方便地为list中的每个元素或map中的每个键值对进行计算或样式输出。这对于区别不大的重复样式是非常有用的。书写格式通常为@each <variable> in <expression> { ... }，其中表达式返回一个列表。块依次为列表中的每个元素求值，该元素被分配给给定的变量名。

$sizes: 40px, 50px, 80px;

@each $size in $sizes {
  .icon-#{$size} {
    font-size: $size;
    height: $size;
    width: $size;
  }
}

$sizes: 40px, 50px, 80px

@each $size in $sizes
  .icon-#{$size}
    font-size: $size
    height: $size
    width: $size

.icon-40px {
  font-size: 40px;
  height: 40px;
  width: 40px;
}

.icon-50px {
  font-size: 50px;
  height: 50px;
  width: 50px;
}

.icon-80px {
  font-size: 80px;
  height: 80px;
  width: 80px;
}

# Mps

您还可以使用@each遍历map中的每个键/值对，书写格式为@each <variable>, <variable> in <expression> { ... }。键被分配给第一个变量名，元素被分配给第二个变量名。

$icons: ("eye": "\f112", "start": "\f12e", "stop": "\f12f");

@each $name, $glyph in $icons {
  .icon-#{$name}:before {
    display: inline-block;
    font-family: "Icon Font";
    content: $glyph;
  }
}

$icons: ("eye": "\f112", "start": "\f12e", "stop": "\f12f")

@each $name, $glyph in $icons
  .icon-#{$name}:before
    display: inline-block
    font-family: "Icon Font"
    content: $glyph

@charset "UTF-8";
.icon-eye:before {
  display: inline-block;
  font-family: "Icon Font";
  content: "\f112";
}

.icon-start:before {
  display: inline-block;
  font-family: "Icon Font";
  content: "\f12e";
}

.icon-stop:before {
  display: inline-block;
  font-family: "Icon Font";
  content: "\f12f";
}

# 解构

如果您有一个数组列表，您可以使用@each来自动为内部列表中的每个值分配变量，书写格式为@each <variable...> in <expression> { ... }。这称为解构，因为变量匹配内部列表的结构。每个变量名都分配给列表中相应位置的值，如果列表没有足够的值，则为null。

$icons:
  "eye" "\f112" 12px,
  "start" "\f12e" 16px,
  "stop" "\f12f" 10px;

@each $name, $glyph, $size in $icons {
  .icon-#{$name}:before {
    display: inline-block;
    font-family: "Icon Font";
    content: $glyph;
    font-size: $size;
  }
}

$icons: "eye" "\f112" 12px, "start" "\f12e" 16px, "stop" "\f12f" 10px;

@each $name, $glyph, $size in $icons
  .icon-#{$name}:before
    display: inline-block
    font-family: "Icon Font"
    content: $glyph
    font-size: $size

@charset "UTF-8";
.icon-eye:before {
  display: inline-block;
  font-family: "Icon Font";
  content: "";
  font-size: 12px;
}

.icon-start:before {
  display: inline-block;
  font-family: "Icon Font";
  content: "";
  font-size: 16px;
}

.icon-stop:before {
  display: inline-block;
  font-family: "Icon Font";
  content: "";
  font-size: 10px;
}

因为@each支持解构，而map被视为列表的列表（数组列表），所以@each的映射支持不需要对映射进行特殊的支持就可以工作。
