# sass:math

兼容性

Dart Sass unreleased |LibSass ✗|Ruby Sass ✗

目前只有Dart Sass支持使用@use加载内置模块。其它的实现用户必须使用全局名称来调用函数。

# math.abs()

  • 包含函数：

    • math.abs($number)
    • abs($number)

  • 返回值： number

  • 用法：

    返回$number的绝对值。 

    @debug math.abs(10px); // 10px
@debug math.abs(-10px); // 10px

    @debug math.abs(10px) // 10px
@debug math.abs(-10px) // 10px

# math.ceil()

  • 包含函数：

    • math.ceil($number)
    • ceil($number)

  • 返回值： number

  • 用法：

    $number向上取整。 

    @debug math.ceil(4); // 4
@debug math.ceil(4.2); // 5
@debug math.ceil(4.9); // 5

    @debug math.ceil(4)  // 4
@debug math.ceil(4.2)  // 5
@debug math.ceil(4.9)  // 5

# math.compatible()

  • 包含函数：

    • math.compatible($number1, $number2)
    • comparable($number1, $number2)

  • 返回值： boolean

  • 用法：

    判断$number1,$number2的单位是否兼容。如果返回true则两数可以安全的进行加法、减法和比较操作。否则，将会产生错误。 

    @debug math.compatible(2px, 1px); // true
@debug math.compatible(100px, 3em); // false
@debug math.compatible(10cm, 3mm); // true

    debug math.compatible(2px, 1px)  // true
@debug math.compatible(100px, 3em)  // false
@debug math.compatible(10cm, 3mm)  // true

    这个函数的全局名称是comparable，但是当它被添加到sass:math模块中时，名称被更改为compatible，以便更清楚地表达函数的功能。

    @debug math.floor(4)  // 4
@debug math.floor(4.2)  // 4

# math.max()

  • 包含函数：

    • math.max($number...)
    • max($number...)

  • 返回值： number

  • 用法：

    返回数字列表中最大的数，数字列表可以使用...。 

    @debug math.max(1px, 4px); // 4px

$widths: 50px, 30px, 100px;
@debug math.max($widths...); // 100px

    @debug math.max(1px, 4px)  // 4px

$widths: 50px, 30px, 100px 
@debug math.max($widths...)  // 100px

# math.percentage()

  • 包含函数：

    • math.percentage($number...)
    • percentage($number...)

  • 返回值： number

  • 用法：

    将无单位数字$number（通常是一个0-1的小数）转化为一个百分比数。 

    @debug math.percentage(0.2); // 20%
@debug math.percentage(100px / 50px); // 200%

    @debug math.percentage(0.2)  // 20%
@debug math.percentage(100px / 50px)  // 200%

    这个函数与$number * 100%相同。

# math.random()

  • 包含函数：

    • math.random($limit: null)
    • random($limit: null)

  • 返回值： number

  • 用法：

    如果$limit是null值，则返回一个0-1的随机小数。 

    @debug math.random(); // 0.2821251858
@debug math.random(); // 0.6221325814

    @debug math.random()  // 0.2821251858
@debug math.random()  // 0.6221325814

    如果$limit是一个大于等于的数，则返回一个1 - $limit的随机数。 

    @debug math.random(10); // 4
@debug math.random(10000); // 5373

    @debug math.random(10)  // 4
@debug math.random(10000)  // 5373

    • round($number)

  • 返回值： number

  • 用法：

    $number四舍五入。 

    @debug math.round(4); // 4
@debug math.round(4.2); // 4
@debug math.round(4.9); // 5

    @debug math.round(4)  // 4
@debug math.round(4.2)  // 4
@debug math.round(4.9)  // 5

# math.unit()

  • 包含函数：

    • math.unit($number)
    • unit($number)

  • 返回值： quoted string

  • 用法：

    返回$number的字符串形式单位。 

    @debug math.unit(100); // ""
@debug math.unit(100px); // "px"
@debug math.unit(5px * 10px); // "px*px"
@debug math.unit(5px / 1s); // "px/s"

    @debug math.unit(100)  // ""
@debug math.unit(100px)  // "px"
@debug math.unit(5px * 10px)  // "px*px"
@debug math.unit(5px / 1s)  // "px/s"

    此函数用于调试；它的输出格式不能保证在Sass版本或实现之间保持一致。
