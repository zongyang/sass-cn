# 布尔运算

与JavaScript等语言不同，Sass使用单词而不是符号来表示布尔运算符。

  • not <expression>，取反操作，返回true或false。
  • <expression> and <expression>，和操作，返回true或false。
  • <expression> or <expression>，或操作，返回true或false。
@debug not true; // false
@debug not false; // true

@debug true and true; // true
@debug true and false; // false

@debug true or false; // true
@debug false or false; // false

@debug not true  // false
@debug not false  // true

@debug true and true  // true
@debug true and false  // false

@debug true or false  // true
@debug false or false  // false

# 真值与否定值

在任何允许true或false的地方，也可以使用其他值。false和null是否定值（falsey），这意味着Sass认为它们表示错误并导致条件失败。其它的值视为真值，所以Sass认为它们是true一样工作，并使条件成功。

例如，如果想检查一个字符串是否包含空格，可使用index($string, " ")。str-index()如果没有找到字符串则返回null，否则返回一个数字。

许多语言不仅仅把false和null否定值。Sass不在其！空字符串、空列表和数字0在Sass中都是真值。
