# 混合器

CSS中的一些东西编写起来有些单调乏味，尤其是使用CSS3和现有的许多供应商前缀。混合器允许您创建希望在整个站点中重用的CSS声明组。您甚至可以传入值，使混合器更加灵活。混合器的一个很好的用法是用于供应商前缀。这里有一个transform的例子。

@mixin transform($property) {
  -webkit-transform: $property;
  -ms-transform: $property;
  transform: $property;
}
.box { @include transform(rotate(30deg)); }

=transform($property)
  -webkit-transform: $property
  -ms-transform: $property
  transform: $property
.box
  +transform(rotate(30deg))

.box {
  -webkit-transform: rotate(30deg);
  -ms-transform: rotate(30deg);
  transform: rotate(30deg);
}

要创建一个mixin，可以使用@mixin指令并给它一个名称。我们已经命名了混合器 transform。我们还在圆括号中使用变量$property，这样我们就可以传入我们想要的任何transform。创建混合器之后，可以将其作为CSS声明使用，以@include开头，后面跟着混合器的名称。
